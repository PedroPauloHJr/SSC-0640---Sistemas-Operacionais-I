Para compilar o programa, temos:
	gcc prog.c -o prog
Para executar o programa, temos:
	./prog
Em seguida, voce deve inserir um numero inteiro que corresponde ao tamanho da Sequencia de
Fibonacci que deseja imprimir na tela. É recomendado inserir uma entrada maior ou igual a 40,
pois é a partir desse instante, que o programa torna-se CPU bound visivelmente.
